# E-Commerce Web Application | Spring Boot

A fully functional, production-deployed E-Commerce web application developed with a strong backend focus using Java and Spring Boot. The system follows clean layered architecture, exposes secure REST APIs, integrates with a MySQL database, and is deployed on Railway Cloud with a live public URL.

Live Application:
https://ecommerce-project-production-be6a.up.railway.app/login.html

Project Overview:
This project simulates a real-world e-commerce platform where users can authenticate, browse products, and interact with the system through REST-based workflows, while admins manage products through protected endpoints.

Core Features:
- Secure user authentication and role-based authorization (Admin / User)
- RESTful API design using Spring Boot
- Admin-controlled product management
- User-side product browsing and cart operations
- Persistent data storage using MySQL and JPA/Hibernate
- Frontend-backend integration via HTTP APIs
- Cloud deployment with environment-based configuration

Technology Stack:
Backend: Java, Spring Boot, Spring Security, JPA/Hibernate  
Frontend: HTML, CSS, JavaScript  
Database: MySQL  
Tools & Cloud: Railway, GitHub, Postman

Deployment & Architecture:
- Deployed as a cloud-hosted Spring Boot application on Railway
- Uses environment variables for database credentials and configuration
- Backend remains live independent of local system
- Follows Controller → Service → Repository layered architecture
- Clean separation of concerns and maintainable code structure

What I Learned:

- Implementing authentication and authorization using Spring Security
- Working with relational databases using JPA and Hibernate
- Deploying and maintaining applications on cloud platforms
- End-to-end development of a full-stack application
