# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application developed using Java Spring Boot for the backend and HTML, CSS, and JavaScript for the frontend. The application is fully deployed on Railway Cloud and accessible through a public URL.

Live Demo: https://ecommerce-project-production-be6a.up.railway.app/login.html

Features:
- User authentication with login and registration
- Role-based access control (Admin and User)
- Product management with admin privileges
- View products and add to cart functionality
- Secure REST APIs using Spring Boot
- Persistent MySQL database integration

Tech Stack:
Backend: Java, Spring Boot, Spring Security, JPA/Hibernate, MySQL  
Frontend: HTML, CSS, JavaScript  
Tools & Platforms: Railway Cloud, GitHub, Postman

Deployment:
- Application deployed on Railway Cloud
- Backend runs independently of local machine
- Uses environment variables for database configuration
- Publicly accessible live URL

Project Structure:

│   ├── controller
│   ├── service
│   ├── repository
│   └── entity
├── src/main/resources
│   ├── static
│   └── application.properties
└── pom.xml

Key Learnings:
- Developed RESTful APIs using Spring Boot
- Implemented authentication and role-based authorization
- Integrated MySQL database using JPA
- Deployed a full-stack application to cloud
- Connected frontend with backend APIs
